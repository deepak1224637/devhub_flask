## 🔧 Features

- ✍️ Admin dashboard to post blogs
- 🔒 Login-secured admin area
- 📚 Static pages for Jobs, Projects, FAQs, etc.
- 🌐 Bootstrap-based UI
- 💾 SQLite database backend

---

## 🛠️ Tech Stack

- Python 3.x
- Flask
- HTML5 + Bootstrap 5
- SQLite (via `sqlite3`)
- Jinja2 (template rendering)
